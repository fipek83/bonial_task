AWS Setup

An EC2 instance of type t2.micro based on a Debian Stretch Image.
A Loadbalancer forwarding incoming requests to the EC2 instance.
The EC2 instance needs to run an Nginx webserver serving one HTML file (just make one up and make the file part of your Github repo). The Nginx server is a Docker container started on the EC2 instance.

Please create a README.md file inside your repository documenting your project.
